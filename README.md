# User Analyzer

A simple Python console project for analyzing users by city and age.

## Features

- Show users from a selected city
- Show users older than selected age
- Show users in selected age range
- Sort users by age
- Interactive menu
- Basic input validation with try / except

## Technologies

- Python
- Git
- GitHub

## How to run

1. Open the project folder
2. Run the file:

```bash
python user_analyzer.py
```

3. Choose an option from the menu:

```text
1 - City report
2 - Age report
3 - Age range report
0 - Exit
```
